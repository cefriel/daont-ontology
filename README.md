# Digital Assets Ontology (DAOnt)

This repository hosts the Digital Assets Ontology (DAOnt).

| Prefix    | IRI |
| -- | ------- |
| dao:  | [https://w3id.org/daont#](https://w3id.org/daont#)  |

## Ontology purpose and scope

The Digital Assets Ontology (DAOnt) addresses the digital assets domain, that is, any hardware, software, data, or service that exists in or interacts with a digital ecosystem and provides functional or informational value.

For designing the ontology, we adopted the [Linked Open Terms (LOT) methodology](https://zenodo.org/records/2539305).
